와인 가이드 나의 취향 나의 평점 와인 지도 시음 방법 홈으로
시음 방법 시음 가이드

눈 (Appearance)

  • 맑기: 맑음 / 흐림 / 탁함
  • 색상:
    • 화이트: 레몬 / 골드 / 호박
    • 로제: 연한 살몬 / 오렌지빛 / 핑크
    • 레드: 루비 / 가넷 / 보라 / 벽돌색
  • 강도: 연함 / 중간 / 진함

코 (Nose)

1차 향 (Primary Aromas)

  • 꽃: 아카시아, 캐모마일, 장미 등
  • 초록 과일: 사과, 구스베리, 배, 모과, 포도
  • 감귤류 과일: 자몽, 레몬, 라임, 오렌지 껍질
  • 핵과류: 복숭아, 살구, 천도 복숭아
  • 열대 과일: 바나나, 리치, 망고, 멜론, 패션푸르트, 파인애플
  • 붉은 과일: 라즈베리, 레드 체리, 딸기, 붉은 자두
  • 검은 과일: 블랙베리, 블루베리, 블랙체리, 블랙커런트
  • 말린/익힌 과일: 무화과, 건포도, 말린 자두, 잼 같은 향
  • 식물성/허브/향신료: 초록 피망, 민트, 후추 등
  • 기타: 부싯돌, 젖은 돌 등

2차 향 (Secondary Aromas)

  • 효모/자가발효: 빵, 토스트, 비스킷, 브리오슈
  • MLF: 버터, 크림, 치즈
  • 오크 숙성: 바닐라, 커피, 초콜릿, 훈제, 코코넛, 삼나무

3차 향 (Tertiary Aromas)

  • 산화 향: 아몬드, 마지팬, 호두, 카라멜, 초콜릿
  • 숙성된 과일 향: 말린 자두, 무화과, 익힌 블랙베리 등
  • 병 숙성 화이트: 꿀, 휘발유, 버섯, 건초, 향신료
  • 병 숙성 레드: 가죽, 흙, 담배, 젖은 잎, 육향

입 (Palate)

  • 단맛 (Sweetness): 드라이 / 오프 드라이 / 미디엄 / 스위트
  • 산도 (Acidity): 낮음 / 중간 / 높음
  • 탄닌 (Tannin): 없음 / 낮음 / 중간 / 높음
  • 질감: 부드러운 / 날카로운 / 떫은 / 입 안을 조이는
  • 알코올 (Alcohol): 낮음 / 중간 / 높음
  • 바디감 (Body): 라이트 / 미디엄 / 풀바디
  • 풍미 강도: 약함 / 중간 / 강함
  • 풍미 종류: 향기 항목과 유사하게 선택
  • 여운 (Finish): 짧음 / 중간 / 김 / 매우 김
