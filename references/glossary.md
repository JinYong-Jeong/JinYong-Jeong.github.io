# Glossary

블로그에서 반복해서 사용하는 용어를 정리합니다.

| Term | Meaning |
| --- | --- |
| LLM | Large Language Model. 대규모 언어 모델입니다. |
| Federated Learning | 데이터를 한 곳으로 모으지 않고 여러 client에서 학습한 결과를 조합하는 학습 방식입니다. |
| PEFT | Parameter-Efficient Fine-Tuning. 적은 수의 파라미터만 조정해 모델을 fine-tuning하는 방법입니다. |
| LoRA | Low-Rank Adaptation. PEFT 계열의 대표적인 adapter 학습 방법입니다. |
| Experiment Log | 실험 설정, 결과, 실패 원인, 다음 action을 재현 가능하게 남긴 기록입니다. |
| Paper Review | 논문의 핵심 문제, 방법, 실험, 한계, 구현 관점을 정리한 글입니다. |
